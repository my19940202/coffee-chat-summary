我想计划在线下摆摊的方式扩展自己的商业收入，目前想到可以做这四类 AI 工具，贴近线下生活场景：
- 老照片修复
- 过世老人 AI 生成视频：数字人视频
- 宠物写真生成
- 老人回忆录生成

下面围绕「技术调研 / 市场规模 / 定价 / 项目规划」做系统整理。

1. 技术调研

1.1 老照片修复  
- **基础能力**：传统图像修复 + 超分辨率，让照片更清晰、去噪点、补全缺失区域。  
  - https://huggingface.co/spaces/sczhou/CodeFormer  
  - https://huggingface.co/spaces/modelscope/old_photo_restoration  
- **视觉大模型补细节**：用通用多模态模型去脑补纹理、光影和服饰细节。  
  - ChatGPT / GPT-4o Vision：https://chatgpt.com/share/69a5804a-ef64-800a-b024-5afeed15d750  
  - 豆包、深度求索（DeepSeek）、Grok 等平台上传图片直接修复  
- **流程建议**：  
  - 线下收照片（手机拍照 / 扫描）→ 在线批量上传 → 模型修复 → 手工挑选几版 → 给用户当场对比确认。  

1.2 过世老人 AI 数字人视频  
- **基础能力**：从多张照片重建人脸 → 语音克隆 / 语音拟声 → 文案生成 → 驱动数字人说话。  
- **可选技术路径**：  
  - 图像：基于头像的数字人（2D）生成，使用脸部驱动 + 口型同步方案。  
  - 语音：用家属提供语音样本（至少 1–3 分钟），做音色拟合，也可用「相似风格」而不做 1:1 拟声。  
  - 文案：大模型根据家属提供的关键记忆点，写祝福、叮嘱或节日问候脚本。  
- **流程建议**：  
  - 线下沟通需求和边界（能接受的「像 / 不像」程度，时长、语气）→ 采集素材（照片、声音、故事）→ 线上生成 → 回传草稿 → 修改 → 输出最终视频。  

1.3 宠物写真生成  
- **基础能力**：宠物 LoRA / DreamBooth 训练 → 一键生成多风格写真（写真照、油画、古风、插画等）。  
- **可选技术路径**：  
  - 使用开源 Stable Diffusion + 宠物 LoRA 模型。  
  - 或调用第三方 API（如 Replicate 等）节省本地算力投入。  
- **流程建议**：  
  - 线下帮助拍 8–15 张清晰宠物照 → 在线训练专属风格 → 当场或次日出图若干套 → 用户选片 + 再生成一轮。  

1.4 老人回忆录生成  
- **基础能力**：语音转文字 + 大模型整理成故事 + 版式排版（图文稿 / 小册子 / PDF）。  
- **可选技术路径**：  
  - 录音：用手机 / 简单录音设备，采集 30–60 分钟访谈。  
  - 文字：ASR（语音识别）转写 → 大模型做时间线整理、主题拆分、口语转书面。  
  - 成品形态：电子版 PDF、小程序 H5 长页、可再衍生短视频配音版。  
- **流程建议**：  
  - 线下和家属/老人约时间访谈 → 录音 → 上传转写 → 大模型写初稿 → 人工润色和核对细节 → 生成排版版式。  

1.5 公共基础能力 / API 供应商  
- 统一的云端推理平台（按量付费，减少本地算力成本）：  
  - https://replicate.com/  
  - https://kie.ai/zh-CN/pricing  
- 后期如量大再考虑：  
  - 自建 GPU / 云服务器，或使用国内厂商推理平台（百度、阿里、腾讯等）。  

2. 市场规模评估（粗粒度）

- **老照片修复**  
  - 适用人群：有大量纸质老照片的中老年家庭、婚纱照影楼、照相馆。  
  - 场景密度：旅游景点、老街、居民区周边、商场人流入口。  
  - 粗略判断：单人服务客单价不高，但高频、转介绍可能性大，可做成“流量入口”项目。  

- **过世老人 AI 数字人视频**  
  - 适用人群：有强情感需求、愿意为纪念仪式感付费的家庭。  
  - 市场特点：低频高客单价，强情绪价值，口碑敏感；更适合作为预约制、工作室级服务，而非大路边“吆喝”的摊位。  
  - 线下摆摊可以侧重「咨询 + 介绍案例」，引导加微信后私聊、后续远程成交。  

- **宠物写真生成**  
  - 适用人群：年轻宠物主人、情侣家庭，宠物店、宠物医院周边。  
  - 场景密度：宠物集市、商场宠物活动、居民区周末市集。  
  - 粗略判断：客单价中等，冲动消费属性强，适合线下摆摊的“快体验 + 即刻出图”模式。  

- **老人回忆录生成**  
  - 适用人群：有时间和精力为长辈做纪念的中年子女；也可以是社区 / 养老院合作项目。  
  - 市场特点：中频中高客单价，决策链条稍长（需要家人一起讨论），更适合通过线下摊位获客，之后预约深度访谈完成交付。  

3. 产品定价（初步假设，后续可根据反馈调整）

- **老照片修复**  
  - 按张计费：如 29–49 元 / 张（简易修复）；多张打包优惠。  
  - 套餐：  
    - 入门包：3 张 79 元  
    - 家庭包：10 张 199 元  
  - 摆摊优惠：现场下单送 1 张对比预览或小尺寸冲印。  

- **过世老人 AI 数字人视频**  
  - 按时长计费：  
    - 30 秒：399–699 元  
    - 1 分钟：699–1299 元（含一次文案修改）  
  - 增值项：  
    - 语音克隆（更像本人）+X 元  
    - 多场景版本（生日版 / 新年版 / 纪念日版）打包价。  
  - 建议策略：现场只报「起价区间」，详细报价放在私聊文档中，配合服务流程说明，降低现场沟通压力。  

- **宠物写真生成**  
  - 单次体验价：  
    - 基础体验：9.9–29.9 元，现场生成 3–5 张低分辨率图（可加水印）。  
    - 正式写真套餐：49–99 元，含 10–20 张高清无水印图。  
  - 会员/复购：  
    - 老客户下次再来，可半价加拍新风格。  

- **老人回忆录生成**  
  - 文字稿套餐：  
    - 精简版（3000–5000 字）：399–699 元  
    - 完整版（8000–15000 字）：799–1499 元  
  - 图文册子（电子版）：在文字稿基础上 +200–400 元，配历史照片整理和排版。  
  - 可以做分阶段收费：先收定金 + 完成初稿后补尾款，降低用户一次性决策压力。  

4. 项目规划与线下注意事项

4.1 项目推进步骤  
- 先完成各工具的 MVP（最低可用版本）：  
  - 老照片修复和宠物写真优先，因为流程短、当场可以看到效果。  
  - 数字人视频、回忆录放在第二阶段，走「预约制 + 远程/线下深度沟通」。  
- 小规模线下试点：  
  - 门口美甲店 / 理发店门口测试（人流稳定、容易沟通）。  
  - 商场内 AI 摆摊 / 周末市集。  
  - 市区艺术区 / 创意市集。  
  - 上海热门旅游景点，主打「游客照片变艺术照 / 老照片数字修复」。  
- 根据试点反馈，调整：  
  - 摆摊物料（展板、价目表、案例展示）。  
  - 产品话术（更贴近用户实际关心的问题）。  
  - 定价和交付时间承诺。  

4.2 线下与用户直接沟通的注意事项  
- **1）隐私与授权**  
  - 清楚说明：所有照片、语音、视频素材只用于当前服务，不会公开、不用于二次训练或商用。  
  - 重要项目（特别是过世亲人数字人视频、老人回忆录）建议让用户签一个简单的「授权与隐私确认单」。  
  - 不主动展示其他客户的素材，案例最好用自己家人 / 虚构示例 / 模型官方 DEMO。  

- **2）情绪与伦理边界**  
  - 面对谈及过世亲人、老人回忆时，注意语气温和、少炫技，多共情。  
  - 明确提示：「AI 无法百分百还原本人，只是用技术帮你多一个纪念方式。」提前打预期，避免后期纠纷。  
  - 遇到明显情绪激动的用户，要懂得适度收缩服务范围，避免引发心理负担或误解。  

- **3）期望管理与交付说明**  
  - 现场说明：成品效果、清晰度、时长、交付时间（如 24 小时内 / 3 天内）和修改次数。  
  - 对于数字人视频、回忆录这类复杂项目，要写一个简单的「服务流程卡片」，让用户拿回去看看。  
  - 避免在现场承诺「无限修改」「想要什么都可以」，而是给清晰边界。  

- **4）付款与售后**  
  - 小金额（照片修复、宠物写真）可以直接现场一次性付款。  
  - 大金额项目建议：定金 + 尾款，结合分阶段交付（初稿 / 终稿）。  
  - 简单约定退款规则：比如「若 X 天内未交付则全额退款」等，写在小卡片或收据上。  

- **5）现场体验与物料**  
  - 准备几个高质量「前后对比 / 成品样例」：老照片修复前后、宠物写真不同风格、回忆录排版截图。  
  - 摆摊空间要整洁，给人“认真做事”的感觉，而不是“随便玩玩的 AI 摊位”。  
  - 准备简单桌牌：扫码加微信、服务介绍、价目表、隐私说明的简短版本。  

- **6）沟通话术与自我保护**  
  - 设计几个固定话术，降低自己第一次线下沟通的焦虑，比如：  
    - 「我们是做 AI 纪念和照片修复的小项目，可以帮你把老照片修得更清晰、更好看。」  
    - 「如果你有一些关于长辈/宠物的故事，我们可以帮你整理成一本小回忆录。」  
  - 遇到不理解 AI 或明显挑刺的人，保持礼貌但不纠缠，把重心放在真正有兴趣的用户身上。  

- **7）合规与场地关系**  
  - 提前确认摆摊地点是否允许拍照、采集用户信息、收款。  
  - 与店铺 / 场地方沟通：可以考虑「抽成 / 固定摊位费 / 互相导流」等合作方式。  
  - 注意当地对人像、逝者形象等内容的相关规定（如不制作违法、敏感、传播性内容）。  

整体思路：先用「老照片修复 + 宠物写真」这样门槛低、体验直观的项目去练线下摆摊和沟通能力，再逐步引入「数字人视频」「老人回忆录」这些更高情感价值和客单价的项目。
