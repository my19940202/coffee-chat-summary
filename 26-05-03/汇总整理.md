## 一、主持人核心叙事

### 1.1 自我定位

「素人」：无名校/大厂光环、无「大结果」，在垂直场景里持续在场，获得扩圈与影响力。

### 1.2 方法论：坚持出摊 + 坚持表达

- **出摊**：固定节奏线下活动（如每周日），风雨无阻。  
- **表达**：出摊产生的素材发到朋友圈等，形成可被搜索、被转述的痕迹。

### 1.3 主要经历节点

小红书被搜到 → 虎嗅 FM/创新节等 → 知乎大场与开放麦（高加好友率、触达出版社与机构）→ 极客公园等报道带来搜索与注册 → 钉钉发布会嘉宾（一线执行同学刷到并紧急邀约）→ 央视、法新社等报道 → 新书与线下活动结缘、直播首发 → 高校 MBA 邀约等，形成**案例撬动案例**的链条。

### 1.4 策略归纳

- 在赛道里**埋伏**，等待第一个机会窗口，再以此为**杠杆**推进下一层合作。  
- **资源撬动资源**（例如重大报道带来的信任，用于后续合作叙事）。  
- 影响力先带来隐性资源，再间接转化为商业结果。  
- **社区**：单靠个人难以完成大节点，需要同伴一起往外冲。

## 三、AI 剧本评估项目 — 行业背景


| 维度        | 要点                                                                                                    |
| --------- | ----------------------------------------------------------------------------------------------------- |
| **公司基因**  | 从内容团队与独立电影起步，电影节有获奖经历；以短剧、广告等商业化项目支撑创作理想。                                                             |
| **短剧与出海** | 短剧热度高；团队去年上线 **50+** 部作品；自称海外短视频侧爆款率约 **26%**，约为部分同行的 **2–3 倍**；已形成相对成熟的产品线。                          |
| **能力栈**   | 覆盖内容、拍摄到后期，将「爆款」从纯玄学尽量拆解为可操作的环节；长期作为乙方服务大厂，熟悉合作中的摩擦与真实痛点。                                             |
| **产品命题**  | AI 降低创作门槛后，产量上升但易出现低质输入与低质输出；市场需要可复用的「好剧本」标准、结构化评分与修改建议。                                              |
| **差异化**   | 垂直行业经验、乙方视角、对评审维度与生产链的理解，被团队视为难以被通用大厂产品替代的部分。                                                         |
| **评估逻辑**  | 结合平台侧专业评审逻辑与用户侧反馈；维度示例：付费钩子、人物与故事张力、不同市场下的文化适配（如迪拜与美国）、客户内部的 **S+ / A / A-** 等；另有团队自设「达不到即难以成功」的红线维度。 |
| **用户与商业** | 当前以 **to B** 专业创作者为主，同时预判未来人人能批量产出剧本；收费形态包含会员与人工深度服务升级。                                               |
| **典型痛点**  | 从业者「做短剧但不爱看短剧」：具备审判能力后审美高于普通观众，面对每周数十部剧本/成片筛选压力，需要高效评估工具。                                             |
| **座谈延伸**  | 评估与真实投放数据如何闭环、品类细分后打分体系是否分化、工具是否会走向「用 AI 筛选 AI 稿」等。                                                   |
