# 初始沟通 - 结构化梳理

> 来源：CoffeeChat 线下活动记录  
> 活动形式：自我介绍 → 提问收集 → 后续分享与回应

---

## 一、活动背景与场域特点

- **场域定位**：一人公司 / 小团队创业者的线下交流场
- **核心价值**：涌现、连接、跨界碰撞，把不同人的智慧经验做连接
- **参与人群**：全职创业者、大学生、在职探索者、投资人等
- **参与规则**：提问可面向个人/多人/全场；被问者本轮不回应，第三轮集中回应

## 五、共性主题与关键词

- **一人公司 / 小而美**：财务、数字化、AI、出海、身心灵
- **出海**：软件、假发、物流、跨境电商、美区平台
- **AI**：产品复制、应用探索、自媒体结合
- **身心灵 / 体验**：颂钵、塔罗、教练、玄学、旅居、线下体验
- **内耗与心力**：多年创业倦怠、启动困难、精力分配
- **连接与涌现**：跨界合作、资源对接、社群价值
